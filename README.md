# Predicting-dry-spells-in-Algadarif
# Problem Description:
 In many rain-fed farming systems, the biggest threat is not always “no rain all season” — it’s gaps in rainfall after planting. When a dry spell hits, seedlings can die before establishing roots. If planting happens at the wrong time, a single dry spell can wipe out an entire field.

The goal of this competition is to build a model that can detect patterns in climate conditions and warn about dry spells within the July–October season

# Project Wokrflow:
1. Installing and Importing Dependencies
2. Reading the training and testing data files
3. Merging the files to construct the datasets
4. EDA
5. Data Preprocessing
6. Feature Engineering
7. Cross Validation
8. Modeling
